No Nama tag Deskripsi
1 < a > membuat link
2 < b > menebalkan teks
3 < button > Mendefinisikan sebuah tombol diklik
4 < br > Memberi baris baru/pindah baris
5 <base> Mendefinisikan URL dasar/target untuk semua URL relatif dalam dokumen
6 <basefont> Membuat atribut teks default, seperti warna, ukuran, jenis font untuk semua teks dalam dokumen
7 <blink> Membuat teks berkedip
8 <body> Mendefinisikan body/isi dokumen HTML, berfungsi untuk menentukan bagaimana isi suatu dokumen ditampilkan di web browsernya. Isi dokumen tersebut dapat berupa teks, gambar, animas, link dan seterusnya
9 <caption> Membuat caption pada tabel
10 <center> Untuk perataan tengah terhadap teks atau gambar
11 <cite> Mendefinisikan kutipan
12 <code> Mendefinisikan sebuah bagian dari kode komputer
13 <col> Mendefinisikan nilai atribut dari satu kolom atau lebih dalam sebuah tabel
14 <colgroup> Menentukan kelompok dari satu kolom atau lebih dalam sebuah tabel untuk performatan
15 <dd> Mendefinisikan deskripsi dari item dalam daftar definisi
16 <del> Untuk memberi garis tengah pada teks/mencoret teks
17 <dl> Mendefinisikan sebuah daftar definisi
18 <dt> Mendefinisikan istilah (item) dalam daftar definisi
19 <formt> Mendefinisikan sebuah form HTML untuk input form
20 <head> Digunakan untuk memberikan informasi tentang dokumen tersebut
21 <hr> Membuat garis horisontal
22 <html> Mendefinisikan root dari suatu dokumen HTML
23 <img > Berfungsi untuk menampilkan gambar pada dokumen HTML
24 <li> Digunakan untuk menampilkan informasi dalam bentuk item daftar
25 <object> meletakkan embed sebuah objek
26 < ol > Mendefinisikan daftar dalam format penomoran
27 < optgroup > Menampilkan beberapa pilihan yang sudah dikelompokkan dalam bentuk sebuah daftar drop-down
28 < option > Menampilkan beberapa pilihan yang berbentuk dalam sebuah daftar drop-down
29 < p > Membuat sebuah paragraf
30 <pre> Membuat teks dengan ukuran huruf yang sama
31 <q> Mendefinisikan sebuah kutipan singkat
32 <a> Untuk memberi garis tengah pada teks/mencoret teks, fungsi tag ini sama dengan tag <del> tetapi tag <s> tidak dianjurkan sebagai gantinya menggunakan tag <del>
33 <body> Mendefinisikan body/isi dokumen HTML, berfungsi untuk menentukan bagaimana isi suatu dokumen ditampilkan di web browsernya. Isi dokumen tersebut dapat berupa teks, gambar, animas, link dan seterusnya
34 <script> Mendefinisikan client-side script
35 <select> Membuat daftar drop-down
36 <strike> Untuk memberi garis tengah pada teks/mencoret teks, fungsi tag ini sama dengan tag <del>
37 <strong> Membuat teks tebal, fungsi tag ini sama dengan tag <b>
38 <style> Mendefinisikan informasi style untuk dokumen HTML
39 <table> Membuat tabel
40 <tbody> Untuk mengelompokkan isi body di dalam sebuah tabeli
41 <td> Mendefinisikan sel di dalam sebuah tabel
42 <textarea> Mendefinisikan sebuah kontrol input multiline
43 <th> Mendefinisikan sel header di dalam sebuah tabel
44 <thead> Untuk mengelompokkan isi header di dalam sebuah tabel
45 <tfoot> Untuk mengelompokkan isi footer di dalam sebuah tabel
46 <title> Membuat judul untuk dokumen HTML
47 <tr> Membuat baris di dalam sebuah tabel
48 <tt > Mendefinisikan teletype text
49 <u> Membuat teks bergaris bawah, fungsi tag ini sama dengan tag < ins > tetapi tag <u> tidak dianjurkan untuk kategori HTML text formatting melainkan termasuk kategori HTML Style
50 <ul> Mendefinisikan daftar dalam format bullet
